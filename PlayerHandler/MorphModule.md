# MorphModule

## Module Function

### module.Init

Parametre : 
player: Player
specificMorphFolder: Folder
noRemovePackage: boolean
Retour :
AUCUN

Description :
Fonction qui quand appelé va appeller dans un ordre spécifique des fonction pour préparé le character du joueurs
